# Statistics for Decision Making Project

## Project Overview
This project analyzes property price data using statistical techniques in Python. The objective is to understand the distribution of property prices and examine the relationship between the number of rooms and property prices.

## Dataset
The dataset used in this project contains property-related information such as suburb, number of rooms, price, distance, land size, building area, and other attributes.

## Tools Used
- Python
- Pandas
- Matplotlib
- SciPy
- Jupyter Notebook

## Analysis Performed
The following statistical techniques were applied:

- Data loading and exploration
- Descriptive statistics
- Mean, median, and mode calculation
- Standard deviation and variance
- Histogram visualization
- Rooms vs Price analysis
- Scatter plot visualization
- Hypothesis testing using T-test

## Conclusion
The analysis shows that the average property price is approximately 1,075,684. Properties with more rooms generally tend to have higher prices. The hypothesis test confirms that the average property price is significantly different from 1,000,000.
